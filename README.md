# 合伙记 - 合作项目财务协作系统

> **在线预览：** [https://maksy5310.github.io/letsFinace/](https://maksy5310.github.io/letsFinace/)

让每一笔合作账目都清晰透明。基于 OpenSpec 规范实现的完整 Demo，包含项目活动管理、自动利润计算、供应商付款对账、发票管理、Excel 导入导出和多用户协作功能。

## 功能特性

| 模块 | 功能说明 |
|------|----------|
| 项目管理 | 创建项目、添加成员、设置分成比例 |
| 收支记录 | 录入收入/支出、按日期范围查询统计 |
| 利润计算 | 自动计算净利润、按比例分配、情景模拟 |
| 供应商对账 | 添加供应商、记录付款、自动对账 |
| 发票管理 | 上传发票（模拟OCR）、查看发票列表 |
| 报表中心 | 月度/年度收支对比、利润走势、分类占比、趋势分析 |
| 用户管理 | 创建用户、角色分配、权限矩阵查看 |
| 导入导出 | 批量导入交易数据（JSON）、导出完整报表 |
| 系统登录 | 权限验证，仅授权人员可使用 |
| 版本记录 | 系统版本提示与更新记录查看 |

## 用户角色

| 角色 | 权限说明 |
|------|----------|
| 管理员 | 拥有全部权限，可管理项目、用户、供应商、发票及所有财务操作 |
| 合伙人 | 可管理项目、成员、交易、利润和报表 |
| 会计 | 可记录交易、查看利润、管理发票、导入导出数据和查看报表 |
| 出纳 | 可记录交易、查看供应商信息、进行付款操作和查看发票 |
| 开发商 | 可创建和编辑项目、记录交易、查看利润和报表 |
| 供应商 | 仅可查看供应商信息和相关发票 |
| 审计 | 可只读查看全部财务数据 |
| 朋友 | 可查看项目概况、利润分配和报表 |

## 项目结构

```
openspec-demo/
├── public/
│   ├── index.html              # Web 前端界面（单页应用）
│   └── demo-data.json          # 模拟数据
├── src/
│   ├── store.js                # 内存数据存储
│   ├── project.js              # 项目服务
│   ├── transaction.js          # 交易服务
│   ├── profit.js               # 利润计算服务
│   ├── supplier.js             # 供应商服务
│   ├── invoice.js              # 发票服务（含模拟OCR）
│   ├── excel.js                # Excel 导入导出服务
│   └── api.js                  # Express REST API
├── tests/
│   ├── unit/                   # 单元测试
│   └── system/                 # 系统测试
├── docs/
│   ├── user-guide.md           # 用户使用说明
│   └── api-reference.md        # API 参考文档
├── server.js                   # 服务入口
├── package.json
└── README.md
```

## 本地运行

```bash
# 安装依赖
npm install

# 启动服务
npm start

# 访问 http://localhost:3000
```

## 测试

```bash
# 运行全部测试
npm test

# 单元测试
npm run test:unit

# 系统测试
npm run test:system
```

## 技术栈

- 原生 HTML/JS（前端）
